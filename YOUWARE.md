# 港日媽媽生活館 (Japan HK Mama) - 專案說明

這是一個為「港日媽媽生活館」設計的高級精簡版 Landing Page。專案採用 React 18, TypeScript, Vite 和 Tailwind CSS 構建，旨在展示創辦人岩本太太嚴選的日本高品質食材，並引導用戶透過 WhatsApp 進行訂購。

## 專案定位與視覺風格

- **風格**：日式簡約、溫暖、健康、高端食材質感。
- **配色**：
  - 背景：米白色/淺木色 (`#FDFBF7`)
  - 主色：深綠/抹茶綠 (`#2D4A22`) - 代表健康藥膳
  - 強調色：日系紅色 (`#C94A47`) - 代表日本元素與和牛
- **字體**：優雅的無襯線體 (Sans-serif)。

## 已實現功能

1. **響應式設計**：完美適配手機與桌面端。
2. **Header (導航欄)**：包含品牌 Logo、選單連結及預留的購物車圖標。
3. **Hero Section (主視覺區)**：大氣的雙欄設計，突顯品牌核心價值。
4. **Founder Section (品牌故事)**：建立創辦人岩本太太的專業形象與信任感。
5. **Service Tags (核心業務)**：展示 B2C 團購、B2B 商務禮籃及體驗式服務。
6. **Product Showcase (產品展示)**：
   - 使用可複用的 `ProductCard` 組件。
   - 整合 WhatsApp 立即訂購功能（帶有預設訊息）。
   - 預留「加入購物車」功能與狀態管理，方便未來升級電商模式。
7. **Footer (頁尾)**：包含社交媒體連結與版權資訊。

## 技術規範與延伸性

- **組件化開發**：`ProductCard` 獨立封裝，數據驅動。
- **狀態管理**：在 `App.tsx` 中預留 `cart` 狀態。
- **路由預留**：結構清晰，未來可輕鬆擴展 `/products/[id]` 與 `/checkout`。

## 開發與構建

- **安裝依賴**：`pnpm install`
- **本地開發**：`pnpm run dev`
- **生產構建**：`pnpm run build`

## 行動端開發規範 (Mobile Rules)

1. **觸控優化**：所有交互元素最小觸控區域為 44px。
2. **安全區域**：支持 iPhone X+ 等劉海屏設備的 Safe Area。
3. **性能優先**：針對行動端網絡與硬件性能進行優化。
4. **🚨 重要**：下載圖片或設備反饋必須使用 `src/utils/mobileFeatures.ts`。
